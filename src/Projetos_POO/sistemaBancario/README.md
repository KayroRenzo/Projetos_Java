# 🏦 Sistema Bancário

Sistema para gerenciar contas e transações bancárias, desenvolvido em Java com POO.

---

## 🎯 Funcionalidades

### 🏦 Contas
- Criar contas (corrente e poupança)
- Depositar dinheiro
- Sacar dinheiro
- Transferir entre contas
- Verificar saldo

### 🛡️ Validações
- Saldo insuficiente (exceção)
- Conta não encontrada
- Validação de valores

---

## 🛠️ Tecnologias

| Tecnologia | Descrição |
|------------|-----------|
| Java 17 | Linguagem de programação |
| POO | Programação Orientada a Objetos |
| Herança | ContaCorrente e ContaPoupanca |
| Polimorfismo | Métodos sobrescritos |
| Exceções | Tratamento de erros |
| MVC | Model-View-Controller |

---

## 🚀 Como Executar

```bash
cd src
javac Main.java
java Main
